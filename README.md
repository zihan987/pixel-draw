# Pixel Draw

这是一个像素绘画的工具，可以在Vercel部署。

在线体验地址：[draw.zihanachai.com](draw.zihanachai.com)

clone本地：
```bash
$ git clone https://github.com/zihan987/pixel-draw.git
```

本地运行指令：
```bash
$ yarn install
$ yarn start
```

Vercel部署命令：
```bash
$ vercel --prod
```
